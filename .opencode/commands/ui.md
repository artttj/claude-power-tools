Use the `design` skill (from the `ui-ux-pro-max` plugin) on the request below. The skill ships searchable databases of styles, palettes, font pairings, charts, and UX guidelines for React, Next.js, Vue, Svelte, SwiftUI, React Native, Flutter, Tailwind, and shadcn/ui. Use it to plan, build, review, or fix UI work — pick a deliberate design direction, avoid AI-default aesthetics, and ship something that looks specific to the product.

If the `design` skill is not installed, tell the user to install it: `/plugin marketplace add nextlevelbuilder/ui-ux-pro-max-skill && /plugin install ui-ux-pro-max`.
